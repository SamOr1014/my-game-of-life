Game Instruction
KeyBoard Control:
WASD: moving the whole board to the corresponding direction
R: Enable the dot drawing mode
IJKL: Moving the selection dot to corresponding direction
E: To draw or undraw the selected grid

Buttons:
Start: Start game;
Stop: Stop game;
Randomise: Random the board;
Clear: Clear the board
Pattern Template: Show special patterns and allow you to copy to the text area
Instruction: Well, you clicked it already Boi

Text Area:
Can allow you to import your own pattern or template pattern from the "Pattern Template" button.
The "COL" and "ROW" input can allow u to assign the pattern at the exact row and column you wish with the submit button!

Mouse:
Drag on the board to draw what ever pattern you like!
You can also control the rules and the game speed on the left side of the game panel. Enjoy!;
